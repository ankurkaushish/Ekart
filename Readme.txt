This is and Ekart project based on java script , in this project basically you (admin) is adding a product , the product will be saved in local storage on browser.
The user can register and then can login and all the products that are added by admin can be viewed and the user can add them to his cart , and buy then.

cart.html refer to admin page where an admin can add products.

login.html refer to the login and register page , where user can register and login.
